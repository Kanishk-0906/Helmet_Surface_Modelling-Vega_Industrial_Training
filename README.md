# Helmet Surface Modeling - Vega Industrial Training

This repository documents my industrial training in advanced surface modeling at **Vega Helmets**, where I developed a series of helmet models using **Siemens NX**. The projects demonstrate a progressive learning workflow, beginning with inner liner surface development and advancing through Class A surfacing to the creation of a complete sewn helmet model.

**Software:** Siemens NX

**Focus Areas:** Surface Modeling • Class A Surfacing • Product Development • Surface Continuity • Industrial Design

---

## Project Progression

The repository is organized to reflect the progression of skills developed throughout the training.

| Project | Description |
|---------|-------------|
| **Helmet 01** | Basic inner liner surface development |
| **Helmet 02** | Advanced inner liner surface with improved freeform geometry |
| **Helmet 03** | Exterior Class A surface development |
| **Helmet 04** | Complete helmet created by integrating the liner and Class A surfaces into a sewn body |
