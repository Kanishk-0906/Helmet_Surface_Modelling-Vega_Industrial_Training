# Overview

This repository showcases the work completed during my industrial training at **Vega Auto Accessories Pvt. Ltd., Belgaum**, where I developed advanced surfacing skills using **Siemens NX**.

The training focused on developing high-quality **Class A** and **liner surfaces** for multiple helmet models while following industry-standard surfacing workflows, maintaining surface continuity, and validating surface quality.

# Surface Modelling Progression

The industrial training followed a structured progression of surface modelling exercises, gradually increasing in complexity and reinforcing industry-standard surfacing practices.

The repository includes:

- Helmet 01 – Liner Surface Development
- Helmet 02 – Liner Surface Development
- Helmet 03 – Class A Surface Development
- Current Helmet Model – Class A & Liner Surface Development

Each stage demonstrates the application of advanced surfacing techniques, continuity management, and surface quality validation.

---

# Siemens NX Techniques

Throughout the project, the following Siemens NX tools and workflows were applied:

- Sketching using custom datum planes
- Spline Creation
- Through Curves
- Through Curve Mesh
- Studio Surface
- Bridge Curves
- Trim Sheet
- Split Body
- Delete Body
- Project Curves
- Surface Analysis (Zebra Analysis)

Surface continuity was maintained using **G0 (Position)** and **G1 (Tangency)** continuity wherever required during surface development.
