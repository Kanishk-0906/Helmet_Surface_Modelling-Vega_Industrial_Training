# Practice Models

This directory showcases representative helmet surfacing exercises completed during my industrial training at **Vega Auto Accessories Pvt. Ltd.**

These models were developed to strengthen proficiency in **Siemens NX Surface Modelling**, with emphasis on surface creation, refinement, and validation using industry-standard workflows.

## What Each Model Includes

Each practice model contains:

- Final surface render
- Wireframe representation
- Surface quality validation (Zebra Analysis)

**Note:** The original Siemens NX CAD files are not included as they are based on proprietary industrial training data. Only representative images are shared to demonstrate the modelling workflow and engineering approach.
